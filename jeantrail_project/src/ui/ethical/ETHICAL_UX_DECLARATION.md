# Ethical UX Declaration

## Objective
Expose ethical behavior to the user through transparent UI elements.

## Constraints
- No dark patterns
- No forced suggestions
- No hidden incentives
- Fully dismissible UI

## Components
- SuggestionCard: Displays reason, transparency, and permanent dismiss.
- TransparencyOverlay: Shows what data was used and not used.
- ExplainThisActionModal: Maps action to decision with a clear explanation.

## Integration
- Integrates with existing suggestion and transparency logic only.
- Does not introduce new intelligence or background tasks.
- Navigation-only actions; no execution.
